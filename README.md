# ligaelite-2026-1689

LigaElite 2026 is a premium, real-time dashboard for Israel's top football league, Ligat HaAl. It offers fans and analysts a sleek, cyberpunk-modern interface with live player stats, detailed team profiles, and smart scoreboards for the 2026 season, delivering a fast, professional UX with dark themes, gold accents, and green highlights.

## Tech Stack
Next.js (React), Node.js, PostgreSQL, Tailwind CSS, Notion API, Sports Data API (e.g., Sportmonks or custom)

## Setup
Run `docker-compose up` to start the infrastructure.
